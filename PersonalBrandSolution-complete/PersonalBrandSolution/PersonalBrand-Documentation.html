Solution Design

🏗️ Architecture Overview

🔷 PersonalBrand.API port 5000
  • 12 Versioned REST Controllers (api/v1/)
  • Repository Pattern — Generic + Specific
  • Service Layer with Memory Caching (1–6hr)
  • EF Core 8 — SQLite (dev) / SQL Server (prod)
  • 12 Domain Entities with soft-delete
  • Global Exception + Request Logging Middleware
  • Swagger / OpenAPI documentation
  • Serilog structured logging (console + file)
  • API Versioning via Asp.Versioning.Mvc
  • CORS configured for MVC origin
🌐 PersonalBrand.MVC port 5001
  • Typed HttpClient — PersonalBrandApiClient
  • Polly Retry (3x exponential) + Circuit Breaker
  • Task.WhenAll() parallel API calls on page load
  • MVC-side MemoryCache (5–60min TTL)
  • Strongly-typed Razor Views (HomeViewModel)
  • Anti-Forgery tokens on all POST forms
  • TempData toast notifications (POST-Redirect-GET)
  • AJAX Q&A filter + Pipeline lead modal
  • GDPR cookie banner · SEO meta + Schema.org
  • Static file 30-day cache headers
📦 PersonalBrand.Shared
  • ApiResponse<T> — unified response wrapper
  • PagedResponse<T> — pagination model
  • All DTOs — PersonaDto, LeadResponseDto, etc.
  • ContactFormDto, NewsletterDto, AddNoteDto
  • LeadStatus constants (new/contacted/proposal…)
  • CacheKeys constants
  • Referenced by both API and MVC projects
🧪 PersonalBrand.Tests
  • xUnit 2.6.6 test framework
  • Moq 4.20 — service/repo mocking
  • FluentAssertions 6.12 — readable assertions
  • EF Core InMemory — real DB simulation
  • Microsoft.AspNetCore.Mvc.Testing
  • TestDbFactory — isolated DB per test
  • MockCache — real + always-miss variants
  • TestData builder — entity factories
  • 107 tests across 4 layers

Getting Started

🚀 Quick Start Commands

▶️ Run API
cd PersonalBrand.API dotnet restore dotnet run # → http://localhost:5000 # → http://localhost:5000/swagger # → SQLite DB auto-created & seeded
▶️ Run MVC
cd PersonalBrand.MVC dotnet restore dotnet run # → http://localhost:5001 # Start API first!
🧪 Run Tests
cd PersonalBrand.Tests dotnet test --verbosity normal # Run with coverage: dotnet test --collect:"XPlat Code Coverage" # Expected: 107 passed, 0 failed
🗄️ EF Migrations
cd PersonalBrand.API # Add migration: dotnet ef migrations add InitialCreate # Apply to DB: dotnet ef database update # Switch to SQL Server: update # ConnectionStrings in appsettings
